To-Do-List App

  • This project is a simple to-do list application built using HTML, CSS, and JavaScript.

  • It allows users to add tasks, mark them as completed, or remove them from the list.

  • The input field lets users type a task, and upon clicking the "Add" button, the task is displayed as a list item.

  • Each task comes with a delete button (represented by a cross symbol) for easy removal.

  • When clicked, tasks can be toggled as completed by applying a "checked" class. The application also uses localStorage, ensuring tasks remain saved even after the page is refreshed.

  • This makes it a lightweight and useful tool for keeping track of tasks dynamically.

  • You can enhance this further by adding animations, filters, or due-date functionality for better task management.
